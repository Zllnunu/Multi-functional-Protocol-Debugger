# 多功能信号调试与分析平台（FPGA + Cortex-M1）

本项目基于 **高云 ACG720 FPGA**，联合 **Cortex-M1 软核处理器**，构建了一套集 **模拟信号采集、波形发生、数字信号测量、多协议串行调试与上位机可视化分析** 于一体的综合性信号调试系统，可用于通信链路验证、嵌入式系统调试、教学实验与信号处理研究。

---

## 系统功能概述

| 模块类别 | 功能说明 |
|--------|---------|
| **模拟信号采集（ADC）** | 双通道 AD9280，最高 25 MS/s；支持 **本地 LCD 实时显示 + 千兆以太网高速数据回传** |
| **模拟信号发生（DAC）** | 基于 DDS + ROM 查表，驱动 MS9708 DAC，输出速率 ≥ 50 MS/s，支持 **≥5 种可调波形** |
| **数字信号测量** | 支持 PWM 频率、占空比、高电平 / 低电平时长测量，典型误差 ≤ 1% |
| **数字信号编码解析** | 支持 **NRZ、RZ、曼彻斯特、差分曼彻斯特** 基础编码格式识别 |
| **串行协议调试（USB-CDC）** | USB → **SPI / I²C / UART / 多路 PWM / CAN** 协议驱动与调试 |
| **上位机可视化工具** | Python + PyQt5 实现波形实时显示、FFT 频谱分析、峰值保持、AC 显示、Autoset 自动调节功能 |


---

##  硬件组成

| 硬件模块 | 型号 / 说明 | 作用 |
|---------|-------------|------|
| FPGA 主控板 | **ACG720** | 系统控制与逻辑运算核心 |
| 模数转换 | **AD9280 ×2** | 双通道高速模拟信号采集 |
| 数模转换 | **MS9708** | 高速可调波形输出 |
| 存储缓存 | **DDR3 (MT41K128M16JT)** | 采集数据帧缓存与高速回传 |
| 以太网接口 | **RTL8211F-CG + RJ45** | 千兆网络数据传输 |
| USB 控制芯片 | **CY7C68013** | CDC 虚拟串口与多协议映射 |
| 串行协议外设 | CH340E / PCF8563T / 24LC64 / ADC128S022 / TJA1050 | 用于 SPI / I²C / UART / CAN 等协议验证 |
| 显示器 | **4.3" MCU-LCD** | 参数设置 & 本地实时示波显示 |
