# 多功能数字/模拟信号调试平台（FPGA + Cortex-M1）

本项目基于 **高云 ACG720 FPGA** 与 **Cortex-M1 软核**，实现了集 **模拟信号采集、模拟信号发生、数字信号测量、多协议串行调试与上位机可视化** 于一体的多功能调试平台，可用于教学演示、嵌入式系统验证、通信链路测试及信号调试分析。

系统功能概述

| 模块 | 功能说明 |
|-----|---------|
| 模拟信号采集（ADC） | 双通道 AD9280，最高 **25MS/s**；支持本地实时显示 & 千兆以太网高速回传 |
| 模拟信号发生（DAC） | MS9708 并口 DAC，最高 **50MS/s**；支持 **≥5 种可调波形**（正弦/三角/方波/锯齿/尖顶余弦） |
| 数字信号测量 | 可测 **PWM 频率、占空比、高/低电平持续时间**（误差 ≤ 1%） |
| 数字信号编码解析 | 支持 **NRZ、RZ、曼彻斯特、差分曼彻斯特** 等基础编码分析 |
| 串行协议转换 | USB-CDC → **SPI / I²C / UART / PWM / CAN** 多协议调试 |
| 上位机可视化 | Python + PyQt5 实现 **实时波形/FFT/峰值保持/AC 显示/Autoset 自动调节** |
